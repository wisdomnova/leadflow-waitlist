# LeadFlow Waitlist - Visual Identity

## Brand Colors

### Primary Color
- **Violet/Purple**: `#745DF3`
  - Used for: Primary buttons, links, accents, interactive elements, hover states
  - Alternative hover: `#6B5FD8` (darker variant for interactions)

### Background Color
- **Off-White**: `#FBFBFB`
  - Used for: Main background, card backgrounds, subtle surfaces
  - Supports clean, minimal aesthetic

### Text & Button Color
- **Dark Navy**: `#101828`
  - Used for: All text content, dark buttons, headings, body text
  - Primary foreground color for readability

## Neutral Palette

### Support Colors
- **Border**: `#E5E7EB`
  - Used for: Dividers, card borders, input borders
- **Secondary Text**: `#6B7280`
  - Used for: Subheadings, descriptions, helper text
- **Hover Overlay**: `#6B5FD8`
  - Used for: Interactive element hover states

## Typography

### Font Family
- **Primary Font**: `Inter`
  - Used for: All text, headings, UI elements
  - Clean, modern sans-serif with excellent readability
  - Variable font support for weight flexibility

## Design Principles

1. **Minimal & Clean**: Use whitespace effectively, avoid clutter
2. **Accessible**: High contrast ratios for text readability
3. **Modern**: Smooth transitions, subtle gradients, clean corners
4. **Consistent**: Uniform spacing, sizing, and color usage
5. **Interactive**: Clear visual feedback on user interactions

## Usage Guidelines

### Color Combinations
- **Primary Action**: Purple (#745DF3) text/background on white
- **Text**: Dark Navy (#101828) on off-white backgrounds
- **Secondary**: Secondary Gray (#6B7280) for less important information
- **Borders**: Light Gray (#E5E7EB) for subtle dividers

### Spacing & Layout
- Base unit: 4px
- Standard padding: 16px (4x base unit)
- Standard margin: 24px (6x base unit)
- Large gaps: 32px+ (8x base unit)

### Typography Scale
- **Headings**: Bold, high contrast, clear hierarchy
- **Body**: Regular weight, 16px base size
- **Small Text**: Secondary color, reduced size for secondary information
